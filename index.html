Game Help

If your launcher gets stuck on "Check Version" or won’t open:

Quick Fix Steps

  1. Run as Administrator (right-click launcher → Run as admin)
  2. If it still fails, use Cloudflare WARP (temporary fix)
  3. After it works, try opening again without WARP
Download Cloudflare WARP

Extra Checks

  • Windows Defender / Antivirus may block the launcher → allow it
  • Restart PC + router once (simple but works)
  • If nothing helps → open a Support Ticket on the official site

How to Recharge Cash / NX

Works outside Brazil too (credit card option).

  1. Go to https://fcabr.net/
  2. Log in to your account
  3. Click the ☰ menu (top right)
  4. Select Donations
  5. Choose your payment method (credit card works outside Brazil)
  6. Pick the amount you want and complete the payment

Create Account

Quick guide (don’t share passwords with anyone).

Steps

  1. Open the official site: fcabr.net
  2. Go to Support / Atendimento (Discord SOS / support panel)
  3. Click Criar conta
  4. Fill: Login, Password, Email
  5. Press Submit

Pro Tips

  • Use a strong password (12+ chars)
  • Save your login in a safe place
  • If you can’t create → open a ticket (Support Ticket)

EU / Outside Brazil Support

If you’re in EU/US and you lag or can’t connect sometimes:

  • Try WARP ON if launcher fails / “Check Version” appears
  • After login works, try playing without WARP (better ping sometimes)
  • Use Ethernet if possible
  • Close downloads + streaming while updating
